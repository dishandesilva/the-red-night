# The Red Knight Game

A browser-based adventure game built with HTML, CSS, and JavaScript as a university selection test project.

## Overview

The Red Knight Game is a light story-driven 2D obstacle adventure where the player controls the Red Knight, avoids traps, and progresses through increasingly difficult stages.

The project focuses on core front-end game development concepts such as:

- Keyboard-based player control
- Sprite animation using image sequences
- Collision and game-over logic
- Progressive level design
- Sound effects and background music integration

## Key Features

- 3 progressively challenging adventure levels
- Character movement with jump mechanics
- Obstacle and trap-based survival gameplay
- Score tracking and level completion flow
- Start, pause, retry, and navigation actions
- Audio feedback for gameplay and events
- Multi-page game flow with login and menu screens

## Gameplay Controls

- Enter: Start gameplay
- Space: Jump

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Bootstrap (for UI layout components)

## Project Structure

- index.html: Login screen
- index2.html: Main game menu
- level1.html: Adventure Level 001
- level2.html: Adventure Level 002
- level3.html: Adventure Level 003
- script1.js: Level 1 gameplay logic
- script2.js: Level 2 gameplay logic
- script3.js: Level 3 gameplay logic
- style.css, style2.css: Main styling files

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Edge, Firefox)
- Local server environment (recommended): XAMPP

### Run Locally

1. Clone or download this repository.
2. Move the project folder into your web server directory (for XAMPP: htdocs).
3. Start Apache from XAMPP Control Panel.
4. Open the game in your browser using your local host path.
5. Begin from index.html.

## Game Flow

1. Login screen
2. Main menu
3. Level 1
4. Level 2
5. Level 3
6. Win or game-over screens with retry options
